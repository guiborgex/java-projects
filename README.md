# ☕ Projetos em Java 

Este repositório reúne vários exercícios e pequenos projetos feitos durante meu estudo da linguagem Java.

Cada pasta contém um projeto separado, com seu próprio código e um README explicando o que o programa faz.

---

## 📁 Lista de Projetos

- Estrutura Básica em Java
- Entrada e Saída de Dados
- Estruturas Condicionais
- Estruturas de Repetição
- Operações com Vetores
- Exercícios de Lógica de Programação
- Programação Orientada a Objetos (POO)

---

## 📌 Sobre os Exercícios

Os códigos foram feitos para praticar:

- Sintaxe da linguagem Java
- Entrada e saída de dados (`Scanner`)
- Estruturas condicionais (`if / else / switch`)
- Laços de repetição (`for`, `while`, `do while`)
- Vetores e matrizes
- Métodos e organização de código
- Programação Orientada a Objetos (classes, atributos e métodos)

## 🎯 Objetivo

O objetivo deste repositório é acompanhar meu aprendizado da linguagem Java e servir como portfólio das atividades e exercícios que venho desenvolvendo durante meus estudos.
